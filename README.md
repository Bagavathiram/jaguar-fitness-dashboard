# jaguar-fitness-dashboard
Real-time fitness studio analytics dashboard using Power BI

 Project Overview
This project is a real-time analytics dashboard built for a fitness studio to track performance, revenue, and customer engagement.

 Problem Statement
The fitness studio lacked a centralized system to monitor business performance, member engagement, and revenue streams. 

Manual tracking made it difficult to:
- Analyze monthly income and expenses
- Compare member attendance between morning and evening batches
- Identify high-performing workout categories
- Track profit margins effectively

This project aims to build an interactive dashboard to provide real-time insights and support data-driven decision-making.

 Features
- Monthly income tracking
- Rent & electricity expense analysis

- Profit percentage calculation
- Morning vs Evening member comparison
- Workout category distribution
- Member growth trends

 Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization Techniques

 Project Structure
- Dataset: Contains raw fitness data
- Dashboard: Power BI (.pbix file)
- Images: Dashboard preview

 Key Findings
- Evening sessions have significantly higher member attendance compared to morning sessions
- The overall profit margin is approximately 58%, indicating strong business performance
- Cardio training generates the highest revenue among all workout categories
- Member growth shows an upward trend with slight fluctuations across months
- Operational expenses like rent and electricity remain consistent, impacting net profit stability


 Insights
- Evening batches have more members than morning
- Profit margin is around 58%
- Cardio training contributes major revenue

 Business Impact
- Helps management optimize batch timings based on demand
- Enables better financial planning and cost control
- Identifies top-performing services to focus marketing efforts
- Improves decision-making using real-time insights
